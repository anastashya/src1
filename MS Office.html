Microsoft Office

sau MS Office este o suită de programe de birou create de firma Microsoft Corporation, care sunt însoțite și de o aplicație specială suprapusă lor pentru accesarea unitară simplă.
Aplicațiile de birou incluse în MS Office sunt:

  1. Microsoft Word (procesor de text)
  2. Microsoft Excel (analiză de business și calcul tabelar)
  3. Microsoft PowerPoint (program pentru prezentări grafice)
  4. Microsoft Access (program pentru baze de date)
  5. Microsoft Outlook (agendă și client de poștă electronică)
  6. Microsoft Frontpage (editor vizual de pagini web)
  7. Microsoft Photo Editor (editor fotografic)

Bineînțeles există și oferte (pachete) de la Microsoft care cuprind numai o parte din aceste programe.

Microsoft Office Word

Microsoft Word este un procesor de text, parte a suitei (grupului) Microsoft Office. Numele întreg este Microsoft Office Word deși popular se numește Microsoft Word, MS Word sau doar Word în unele cazuri. Prima lui versiune datează din 1983 fiind inițial disponibil sub numele de Multi-Tool Word pentru Xenix systems. În 1985, Microsoft Word a fost lansat și pe (atunci noua) platformă Apple Macintosh, iar în 1989 pentru Windows, Microsoft continuând să suporte ambele platforme până astăzi. În 1992-1993, Microsoft a oferit și o versiune de Word pentru sisteme Unix (mai precis distribuția SCO), ofertă care însă a fost de foarte scurtă durată. Versiunea cea mai recentă este Word 2013 pentru Windows, respectiv Word 2008 pentru sisteme Macintosh (folosind Mac OS X, atât pe arhitectură Intel cât și PowerPC). Formatul nativ a fost .doc, de la versiunea 1 până la Word 2003 devenind de la Word 2007 .docx ( extensia ".docx" este folosită în noul standard internațional Office Open XML pentru documentele Office).
Există și posibilitatea transformării unei pagini scanate, a unei fotografii sau a unui fișier PDF în document Word. Există foarte multe aplicații destinate pentru a realiza acest lucru, care poartă numele OCR (Optical Character Recognition – recunoaștere optică a caracterelor). Există și servicii online care îndeplinesc aceeași sarcină. Eficiența unui astfel de program este dată de abilitatea de a nu interpreta greșit anumite caractere, cuvinte sau pasaje din documentul inițial. De asemenea, multe dintre aceste aplicații oferă opțiunea de a selecta limba în care a fost redactat fișierul PDF sau fotografia care va fi transformată document editabil.

Microsoft Office Excel

Microsoft Excel este un program de calcul tabelar. Astăzi este cel mai utilizat software pentru calcul tabelar.
Excel aparține grupei de programe Microsoft Office și este disponibil atât pentru Microsoft Windows, cât și pentru Mac OS.
Fiind o aplicaţie din pachetul Microsoft Office, Microsoft Excel prezintă o interfaţă asemănătoare cu editorul de text Microsoft Word având aceeaşi organizare a sistemului de meniuri şi a barelor de instrumente. Fiecare foaie de calcul conține coloane (columns) și rânduri (rows) care intersectate formează celule (cells). Coloanele sunt etichetate cu litere în ordine alfabetică (A,B,C,...,Z;AA,AB,AC,...,AZ;BA,BB,BC s.a.m.d.), iar rândurile sunt numerotate crescător (1,2,3,...). Fiecare celulă are o adresă unică formată din litera coloanei și numarul liniei. De exemplu: A1, B5, C12. Putem să ne referim și la un grup de celule indicând celula din colțul din stânga sus și celula din colțul din dreapta jos. De exemplu: (A1:H5).
